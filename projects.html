ML - Enabled Dashboard for MHRD Schemes

AI-powered educational analytics and forecasting platform.

View Project View Code

Problem: Educational data from different MHRD schemes was scattered across multiple sources, making it difficult for administrators to analyze trends or make predictions about enrollment and policy outcomes.

Solution: Developed a machine learning powered dashboard that collects and processes educational datasets to generate insights and predictions about scheme performance.

Tech Stack: Python, Pandas, Scikit-learn, React, REST APIs

Key Features:

  • Enrollment prediction using regression based ML models.
  • Interactive dashboard for visualizing scheme performance.
  • Automated data cleaning and preprocessing pipelines.
  • Real-time charts and analytics for policy monitoring.

Impact: Helps decision makers quickly understand education trends and take data-driven actions instead of relying on manual reports.

SarcasmAI

Sarcasm detection system using NLP.

View Project View Code

Problem: Detecting sarcasm in text is difficult for traditional NLP models because sarcasm often depends on context and tone.

Solution: Built a machine learning based NLP system that analyzes sentence structure and linguistic patterns to detect sarcasm in text.

Tech Stack: Python, NLTK, Scikit-learn, NLP Techniques

Key Features:

  • Text preprocessing including tokenization and stopword removal.
  • Feature extraction using TF-IDF and linguistic patterns.
  • Classification model trained to identify sarcastic statements.
  • Confidence score showing probability of sarcasm.

Impact: Achieved around 87% accuracy in sarcasm detection, demonstrating the effectiveness of machine learning in understanding complex language patterns.
